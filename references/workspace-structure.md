# Neo Workspace Structure — Complete Specification

## Design Principles

1. **Agent-first**: Agents are primary citizens. Ontology is substrate.
2. **Spec before build**: Define what agents should do before building ontology.
3. **Human role: spec & validate**: Humans specify intent and validate results.
4. **Rules vs Skills**: Rules = deterministic YAML. Skills = judgment-requiring markdown.
5. **Self-containment**: Everything an agent needs lives inside the workspace.
6. **Executable ontologies**: Python classes, testable and benchmarkable.
7. **Autoresearch-compatible**: One modifiable workspace, read-only eval.

## Test Tiers

### Unit Tests (Free, Every Commit)

Test rule logic without LLM. Pure Python assertions against YAML rules.

```python
def test_temperature_threshold():
    rule = load_rule("comfort-temperature-range")
    assert rule.evaluate({"zone_temp": 19}) == "flag_comfort_issue"
    assert rule.evaluate({"zone_temp": 23}) is None
```

### Structural Tests (~$0.01/test, Every PR)

Run the agent, then check output structure deterministically.

```python
def test_agent_produces_diagnosis():
    result = run_agent("diagnose warm room 812A")
    assert "root_cause" in result
    assert "recommendation" in result
    assert len(result["recommendation"]) > 0
```

### Semantic Tests (~$0.10/test, Nightly)

Use LLM-as-judge to evaluate response quality.

```python
def test_diagnosis_quality():
    result = run_agent("diagnose warm room 812A")
    grade = llm_judge(result, criteria=[
        "identifies the correct root cause",
        "recommendation is actionable",
        "uses domain terminology correctly"
    ])
    assert grade.score >= 0.8
```

def split_passages(source_text: str, ratio: float = 0.8) -> tuple[list[str], list[str]]:
    """Deterministic hash-based 80/20 split of source material."""
    paragraphs = [p.strip() for p in source_text.split('\n\n') if p.strip()]
