# Agents

## COSTAR Framework

**CO (Curate & Organize):** Neo.Architect builds and maintains the finance and banking ontology.
**STAR (See, Think, Act, Reflect):** Consumer agents use the ontology to solve problems.

## Active Agents

| Agent | Role | Domain |
|-------|------|--------|
| fraud-detector | Analyze loan packages for fraud indicators | Bank statement verification |

## Routing

- "Is this fraudulent?" / "Check this package" / "Verify" → **fraud-detector**
- "What entities exist?" / "Show ontology" → **Neo.Architect** (meta)
